📊 Superstore Power BI Dashboards Overview

This project is a Business Intelligence analysis of the Superstore dataset using Power BI.
It delivers actionable insights on sales, profitability, and customer behavior to support executive decision-making.

🔹 Dashboards
1.	Executive Sales Dashboard
•	Revenue overview by region and category
•	Monthly sales trends
•	Top-performing categories

2.	Executive Profit Dashboard
•	Profitability analysis by region and product
•	Discount impact on profit margins
•	Identifying loss-making segments

3.	Customer Insights Overview
•	Customer segmentation
•	Top customers by revenue
•	Repeat purchase patterns

🛠 Tools & Technologies
•	Power BI Desktop – Dashboard visualization
•	Power Query – Data cleaning & transformation
•	DAX – Measures & calculated fields

🎯 Business Objective
To transform raw retail data into insights that drive business strategy, 
helping executives make data-driven decisions about sales, profits, and customer management.

📌 How to Use
Download the .pbix file from the dashboards folder.
Open it in Power BI Desktop.
Explore interactive visuals, filters, and insights.
